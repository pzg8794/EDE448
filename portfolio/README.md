# Portfolio

The final portfolio is worth 40% of the course grade and is due August 7, 2026.

## Assignment Core

Create a Communication and Positive Behavioral Support Strategies and Resources Portfolio with at least 10 resources/supports. Each resource should include 1-2 pages of content explaining:

- what the strategy/resource is
- how it supports communication, social-emotional learning, or behavior
- when and why it would be used in practice
- relevant sources, references, or websites
