# Local Reading Files Index

This index tracks the reading files that exist in the local course folder but are intentionally not committed to the public GitHub repo.

## Why They Are Not In GitHub

The `Read/` files are mostly PDFs, course packets, textbook excerpts, library exports, and web downloads. They stay local because the repo is public. We track their filenames and how they support the work, but we do not upload the raw files.

## How To Use This

1. Pick the readings for the module.
2. Record the selected readings in the module task tracker.
3. Add the citation to the relevant assignment draft.
4. Keep the PDF/source local.
5. If a source is public web content, link to the public URL instead of uploading a copy.
